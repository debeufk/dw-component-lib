## Contributing

If you are looking to contribute to this package than this page contains all the needed information.

### Prerequisites
 - Node.js (recomended version v12 LTS)
 - Yarn (recomended version v1.22.15)

### Minimal path to awesome
Use the steps below to install and run the code.
 1. Pull the code from [GitHub](https://github.com/dlw-digitalworkplace/dw-component-lib)
 2. Run `yarn install` in the root folder of the project
 3. Run `yarn build` in the root of the folder
 4. Run `yarn start` in the root of the folder.
 5. Choose an application you want to start
    - @dlw-digitalworkplace/dw-component-lib - starts the storybook
	  - @dw-react-app/demo-spfx-app - runs a demo SPFx project which uses the component library

> **Warning**: Errors might occur when using `yarn install` in PowerShell. Best to use CMD for installation purposes.
