# Introduction

`dw-component-lib` is a collection of packages designed to help you in developing React and/or TypeScript projects.

The currently included packages are:

- **dw-react-controls** - a collection of reusable `React` components, written with and for
  Microsoft's [Fluent UI](https://developer.microsoft.com/en-us/fluentui).
- **dw-react-utils** - utilities which can be used to simplify common tasks. Includes utility classes, as well as
  custom React hooks.
- **dw-refresh-cache** - caching library which allows you to ask for the cached data and automatically fetch and notify
  when receiving new data in the background.
- **taxonomyprovider-...** - Taxonomy providers which can be used in the `TaxonomyPicker` control.
  - **sharepoint** - receives its data from the `SP.Taxonomy.js` API. This can be used when hosting on a SharePoint page.

## Installation

To install any of the packages, add it using your favorite package manager.

```shell
npm install -S @dlw-digitalworkplace/dw-react-controls
```

or, using `yarn`:

```shell
yarn add @dlw-digitalworkplace/dw-react-controls
```
