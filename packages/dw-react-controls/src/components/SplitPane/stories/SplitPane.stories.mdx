# SplitPane

The `SplitPane` can be used to split the screen into two parts.
The content of the left and right pane can be set using the render functions.
Use the `initialSizes` prop to set the initial sizes of the panes in percentage.
You can also provide a minimum width of the panes in percentage.

### Basic usage

Provide the `initialSizes` prop and two render functions to get started.

	<Story story={stories.Basic} name="Basic usage" />
</Canvas>

### Minimum width

You can provide a minimum width for the panes in percentage.
When doing so, the `SplitPane` will not allow the panes to be smaller than the minimum width.

	<Story story={stories.MinWidth} name="Minimum widths" />

### Fixed panes

You can set the `fixed` prop to `true` to make the panes fixed.
