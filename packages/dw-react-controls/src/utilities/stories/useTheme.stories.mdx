# useTheme (deprecated)

> ⚠️ The custom `useTheme` hook is **deprecated** in favor of `@fluentui/react`'s version.
> Use `import { useTheme } from "@fluentui/react;` when possible.

The `useTheme` hook provides access to the currently applied theme.

## Usage

```tsx
const theme = useTheme();

// or

const theme = useTheme("MyScope");
```
