# useId

The `useId` hook allows you to generate a unique id for a component.

## Usage

```tsx
const id = useId();

// or

const { customId } = props;
const id = useId(customId);
```

## Example

```tsx
const MyComponent = () => {
	const id = useId();

	return <div id={id}>Hello world</div>;
};
```
