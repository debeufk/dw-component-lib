# dw-refresh-cache

![dw-refresh-cache package version](https://img.shields.io/github/package-json/v/dlw-digitalworkplace/dw-component-lib?filename=packages%2Fdw-refresh-cache%2Fpackage.json&label=%40dlw-digitalworkplace%2Fdw-refresh-cache&logo=github)

The `@dlw-digitalworkplace/dw-refresh-cache` package provides a caching library which allows you to ask for cached data and automatically fetch and notify when receiving new data in the background.

## Installation

```shell
yarn add @dlw-digitalworkplace/dw-refresh-cache
```
