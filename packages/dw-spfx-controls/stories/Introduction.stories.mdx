# dw-spfx-controls

![dw-spfx-controls package version](https://img.shields.io/github/package-json/v/dlw-digitalworkplace/dw-component-lib?filename=packages%2Fdw-spfx-controls%2Fpackage.json&label=%40dlw-digitalworkplace%2Fdw-spfx-controls&logo=github)

The `@dlw-digitalworkplace/dw-spfx-controls` package contains reusable controls which can be used in any **SharePoint Framework** application.

These controls are designed specifically for use within an SPFx context, and (can) have dependencies on specific SPFx libraries.

## Installation

```shell
yarn add @dlw-digitalworkplace/dw-spfx-controls
```
